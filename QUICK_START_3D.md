# 🚀 GUIDE RAPIDE - Convertir FBX en GLB pour VO390

## ⏱️ Durée: 5 minutes

---

## 📋 Résumé

Tu dois convertir 4 fichiers FBX en GLB en utilisant un convertisseur en ligne, puis les placer dans le dossier `3D - Models/VO390/`.

---

## ✅ Étapes

### 1️⃣ Ouvrir le convertisseur Aspose (Recommandé)

👉 **Clique ici:** https://products.aspose.app/3d/conversion/fbx-to-glb

*(Pas d'inscription, pas de limite, gratuit)*

### 2️⃣ Sélectionner les fichiers FBX

1. Clique sur **"Upload Files"**
2. Navigue vers: `c:\Users\asamu\Desktop\Configurateur V2\3D - Morth Targets\VO390\`
3. Sélectionne les 4 fichiers:
   ```
   ✓ VO390_1.fbx
   ✓ VO390_2.fbx
   ✓ VO390_3.fbx
   ✓ VO390_4.fbx
   ```
4. Clique sur **"Open"** ou **"Upload"**

### 3️⃣ Sélectionner le format de sortie

1. Sélectionne **"GLB"** dans le dropdown (en haut à droite)
2. Le format doit être: **glTF Binary (.glb)**

### 4️⃣ Convertir

1. Clique sur **"Convert"** ou **"Convert Now"**
2. Attends que la conversion soit terminée (2-10 secondes)

### 5️⃣ Télécharger les fichiers

1. Les fichiers convertis s'affichent
2. Clique sur chaque lien de téléchargement:
   ```
   ✓ VO390_1.glb
   ✓ VO390_2.glb
   ✓ VO390_3.glb
   ✓ VO390_4.glb
   ```

### 6️⃣ Placer les fichiers dans le bon dossier

1. Ouvre le dossier: `c:\Users\asamu\Desktop\Configurateur V2\3D - Models\VO390\`
2. Copie/Colle les 4 fichiers GLB téléchargés ici
3. Vérifie que les fichiers sont présents:
   ```
   c:\Users\asamu\Desktop\Configurateur V2\3D - Models\VO390\
   ├── VO390_1.glb ✓
   ├── VO390_2.glb ✓
   ├── VO390_3.glb ✓
   └── VO390_4.glb ✓
   ```

### 7️⃣ Tester dans le configurateur

1. Ouvre **index.html** dans le navigateur
2. Sélectionne le modèle **"VO390"** dans le dropdown
3. Sélectionne **"1 Cuve"**
4. Tu devrais voir le modèle 3D s'afficher avec:
   - 🎬 Une animation 3D interactif
   - ↩️ Bouton "Réinitialiser la vue"
   - 📷 Bouton "Télécharger l'image"
   - 🖱️ Possibilité de tourner/zoomer avec la souris
