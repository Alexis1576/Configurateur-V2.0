# 📂 Dossier pour les modèles 3D VO390

Placez les fichiers GLB convertis ici:

```

├── VO390_1.glb  (1 cuve)

└── VO390_4.glb  (4 cuves)
```

## 📋 Processus de conversion

1. Convertissez les fichiers FBX en GLB using:
   - **Aspose 3D Converter** (Recommandé): https://products.aspose.app/3d/conversion/fbx-to-glb
   - **AnyConv**: https://anyconv.com/en/fbx-to-glb-converter/
   - **CloudConvert**: https://cloudconvert.com/fbx-to-glb

2. Téléchargez les fichiers GLB

3. Placez-les dans ce dossier

4. Testez dans index.html en sélectionnant VO390

## ✅ Vérification

Vérifiez que les fichiers sont présents:
```powershell
Get-ChildItem -Path "3D - Models\VO390" -Filter "*.glb"
```
